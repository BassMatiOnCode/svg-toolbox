Textbox

The Textbox provides an SVG g container with a rect and one or more text elements:

const r = new rect( "Textbox content string", x, y );
target.append( r.svgElement );

This creates an SVGGElement, containing an SVGRectElement and an SVGTextElement. The rectangle provides a border, the text element shows the string in the first argument, both centered at the provided coordinates x and y. The size of the border rectangle is calculated from the extent of the text content, but can also be given explicitly:

const r = new rect( "Textbox content string", x, y, w, h );
target.append( r.svgElement );

const r = new rect( [ "This textbox", "contains", "multiple lines of text", { x : 200 , y : 500 } );
target.append( r.svgElement );

An array of strings is translated into an array of SVGTextElements. The center position can also be given as plain object in the members x and y.

content = [ "This", "is the textbox", "content" ];
new rect( content, x, y );
new rect( content, x, y, w, h, attributes, options ); new rect( content, position, size, attributes, options );
