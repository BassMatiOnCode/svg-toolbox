Point-Slope Form of a Line Equation

A non-vertical line can be describe by the slope m and the coordinates (x1,y1) of any point on the line:

y=y1+m(x−x1) y−y1=m(x−x1)

Here we can easily see the slope relation:

y−y1 x−x1 = m

 

If you set x or y equal to zero, you can easily find the interception points

yy0 = 1 xx0 = 1
