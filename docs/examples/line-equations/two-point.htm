Two-Point Form of a Line Equation

The two-point form of a line equation is based on intersecting lines:

It says that

|AC| |SC| = |BD| |SD|

Translated into a cartesian coordinate system:

From Thales' intercept theorem follows:

y−y1 x−x1 = y2−y1 x2−x1

Solved for y:

y = y2−y1 x2−x1 (x−x1 ) + y1

This equation requires x2−x1≠0.

With m= y2−y1 x2−x1 as slope or gradient follows:

y = m(x−x1 ) + y1

Going back to

y − y1 = y2−y1 x2−x1 (x−x1 )

Then clearing the denominator and subtracting the right side:

(y−y1) (x2−x1) − (y2−y1) (x−x1) = 0

we get a form that is also valid for x2−x1=0.

This form is not symmetric in the two given points, but a symmetric form can be obtained by re-grouping the constant terms:

( y1 − y2 ) x + ( x2 − x1 ) y + ( x1 y2 − x2 y1 ) = 0
