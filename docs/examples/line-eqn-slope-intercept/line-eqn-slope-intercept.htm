Slope-Intersect Form of a Line Equation

A non-vertical line can be defined by its slope m and its y–axis intercept y0:

y = mx + y0

with

m = − y0 x0

If the line is non-horizontal, the line can be defined by slope and x–intersect x0:

y = m ( x − x0 )

These forms can be converted into the form

ax + by + c = 0

by using

m=−ab x0=−ca y0=−cb
