Lines Intersection

Figure 1 shows two lines, defined by the point P1 to P4, intersecting at the point P:

We use the two-point form of a line equation:

y = y2−y1 x2−x1 (x−x1 ) + y1

We a looking for an x where y for both lines is identical:

y2−y1 x2−x1 (x−x1 ) + y1 = y4−y3 x4−x3 (x−x3 ) + y3

By introducing the slope m:

m1 (x−x1 ) + y1 = m2 (x−x3 ) + y3 m1x − m1x1 + y1 = m2x − m2x3 + y3 m1x − m2x = m1x1 − m2x3 + y3 − y1 x ( m1 − m2 ) = m1x1 − m2x3 + y3 − y1

we get an equation for x:

x = 1 m1 − m2 ( m1x1 − m2x3 + y3 − y1 )

Note that the coordinates of the point P in figure 1 are actually calculated with the equation.
